# Лабораторна робота №7  
## Обробка IO/мережевих помилок та патерн Retry

### Мета
Реалізувати обробку винятків та патерн Retry з експоненціальною затримкою для IO та мережевих операцій.

### Реалізовано
- FileProcessor — імітація IO-помилок.
- NetworkClient — імітація мережевих помилок.
- RetryHelper — універсальний механізм повторних спроб.
- shouldRetry — контроль дозволених типів винятків.
- Демонстраційні приклади у Program.cs.
