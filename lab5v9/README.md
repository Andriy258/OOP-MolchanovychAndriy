# Lab5v9: Composition + Generics + Exceptions

## Опис задачі

Модель предметної області: оцінювання тестів.

- Сутності:
  - `Test` – тест із переліком запитань та правильних відповідей.
  - `Answer` – відповідь на конкретне запитання (композиція `Test → List<Answer>`).
  - `Student` – студент, який має відповіді на тести.
- Узагальнений компонент:
  - `Repository<T>` для зберігання студентів (`Repository<Student>`).
  - Власні методи `MaxBy`/`MinBy` для роботи з колекціями результатів.
- Обчислення:
  - Кількість правильних відповідей.
  - Відсоток правильних відповідей.
  - Середній відсоток по групі.
  - Пошук студента з максимальним результатом.
- Винятки:
  - `InvalidAnswerException` при некоректному індексі питання у відповіді.
  - `NotFoundException` у репозиторії.
