# **SR12 — PLINQ: дослідження продуктивності та потокобезпечності**

## **Тема**
Дослідження продуктивності Parallel LINQ (PLINQ) у порівнянні зі звичайним LINQ та аналіз потокобезпечності при паралельній обробці даних.

---

## **Мета**
- Порівняти час виконання LINQ та PLINQ на великих наборах даних.
- Дослідити, коли PLINQ дає прискорення, а коли — ні.
- Продемонструвати потенційні помилки та побічні ефекти при паралельних обчисленнях.
- Навчитись використовувати `Stopwatch`, PLINQ (`AsParallel()`), потокобезпечні механізми (`lock`, `Interlocked`).
